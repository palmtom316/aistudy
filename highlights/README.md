# highlights/

PDF 教材划线缓冲区。PDF++ 插件在 PDF 里选字后"copy as markdown"粘贴到这里，每本书一个文件。

`study-extract` skill 读这里的划线 → 按主题聚类 → 生成 `notes/` 下的 note 草稿（带 Descriptors + source 回链）→ `anki-export.sh` 扫 Descriptors 成卡。

**划线 ≠ 卡**。划线只是"这段重要"，要成卡必须先经 `study-extract` 结构化成 `概念:: 描述子 → 值`（SPEC §3.2）。本目录是缓冲，不是终点。

文件 schema 见 `templates/highlight.md`。
