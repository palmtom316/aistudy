# 建造师 Descriptors 词典

> SPEC §3.2 受控词汇表。`study-outline` 建 note 时从此取描述子；`anki-export.sh` 校验 `::` 行的 key 是否在本表内，词典外一律拒收并跳过。
> anki-export 取本文件 `## 受控描述子` 区块内每行 `key :: 用途` 的 key 部分做白名单。

## 受控描述子

```
定义 :: 术语/概念的官方定义
组成 :: 设备/系统的构成部件
分类 :: 按某维度的类别划分
作用 :: 部件/工序的功能与目的
工序 :: 施工或操作的先后步骤链
参数 :: 规范规定的数值/偏差/技术指标
要求 :: 技术要求或合规条件
规范条文 :: 对应国家/行业规范原文及条款号
```

## 各描述子用法与示例

### 定义
术语的官方/教材定义。
- 行式：`定义:: 绝缘电阻 → 加直流电压后流过的泄漏电流对应的电阻`
- 卡：正=术语-定义，反=定义正文

### 组成
设备或系统的构成部件清单。
- 行式：`组成:: 变压器 → [铁芯, 绕组, 油箱, 冷却装置]`
- 适于成套设备、配电装置等"由哪些部分构成"的知识点

### 分类
按某维度划分的类别。
- 行式：`分类:: 电缆 → [按电压: 低压/高压, 按绝缘: 油浸/干式]`

### 作用
部件或工序的功能目的。
- 行式：`作用:: 接地装置 → 将故障电流泄入大地，限制接触电压`

### 工序
施工/操作的先后步骤链（与 procedure-flow 子模板呼应）。
- 行式：`工序:: 电缆敷设 → 测量放线→开挖缆沟→铺砂垫层→下缆→铺盖板→回填→标识`

### 参数
规范规定的数值、偏差或技术指标。
- 行式：`参数:: 绝缘电阻测量 → 0.6/1kV 电缆 ≥ 0.5MΩ；高压电缆 ≥ 10MΩ/km`
- 数值必须可追溯到教材/规范页码（note 的 source 字段）

### 要求
技术要求或合规条件（区别于参数：要求偏定性，参数偏定量）。
- 行式：`要求:: 电缆埋深 → 一般地段 ≥0.7m；穿越农田 ≥1m`

### 规范条文
对应的规范原文及条款号。
- 行式：`规范条文:: 电缆埋深 → GB 50168-2018 §5.1.3`
- 法规类 note 必须同时填 frontmatter `effective_date`

## 约束

- 描述子 key 必须用上表 8 个之一，**禁止自造**（如"特点""意义"等一律拒收）。
- 一篇 note 的 `## Descriptors` 区块至少 1 行，建议 3-6 行覆盖该知识点主要记忆点。
- `→` 后的值若含数值/条文，note 的 `source` 必须能回链到出处（§5.2 信任链）。
